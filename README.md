# nixpkgs-fileuni

Standalone Nix repository for FileUni CLI and GUI packages.

This repository is generated by the FileUni release workflow.
It is intended for direct end-user installation and does not target the upstream `nixpkgs` repository.

## Supported Environments

- NixOS
- Linux with Nix
- WSL with Nix
- macOS with Nix (CLI only)

## Packages

- `fileuni`: CLI package for Linux and macOS
- `fileuni-gui`: GUI package for Linux (`x86_64-linux`, `aarch64-linux`)

## Quick Start

Install the CLI package with flakes enabled:

```bash
nix profile install github:FileUni/nixpkgs-fileuni#fileuni
```

Install the GUI package on Linux:

```bash
nix profile install github:FileUni/nixpkgs-fileuni#fileuni-gui
```

Run without installing:

```bash
nix run github:FileUni/nixpkgs-fileuni#fileuni -- --help
nix run github:FileUni/nixpkgs-fileuni#fileuni-gui
```

If you are pinned to an older commit without `flake.lock`, use the latest branch ref:

```bash
nix run github:FileUni/nixpkgs-fileuni/main#fileuni -- --help
nix run github:FileUni/nixpkgs-fileuni/main#fileuni-gui
```

You can also force a fresh fetch:

```bash
nix run --refresh github:FileUni/nixpkgs-fileuni#fileuni -- --help
nix run --refresh github:FileUni/nixpkgs-fileuni#fileuni-gui
```

## Legacy Nix

If you are not using flakes:

```bash
nix-env -f https://github.com/FileUni/nixpkgs-fileuni/archive/refs/heads/main.tar.gz -iA fileuni
nix-env -f https://github.com/FileUni/nixpkgs-fileuni/archive/refs/heads/main.tar.gz -iA fileuni-gui
```
